这是一个前端请求的 mock 服务。

## 想要解决的问题

前端开发中依赖后端的接口数据，为了在开发阶段消除前后端的依赖，所以希望开发一个 mock 服务来解决以下问题。

1. 前端和后端约定好接口字段，前后端可以同步工作。
2. 前端录入 mock 数据，形成文档，易于维护。
3. 前端可模拟异常情况，便于接口测试。
